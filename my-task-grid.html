My Task Grid

Task management app built with React

Javascript | React
View Project View Code

Task Management App, built with Javascript and React. Designed to be an easy to use and convenient way to manage a todo list at work.

Tasks are divided into 4 category lists.

All completed tasks are stored in the 'Completed' panel, and can easily be brought back into their respective lists.

Each task has a 'Defer' option, which will hide it from view temporarily, to make it easier to prioritise tasks. Deferred tasks will reappear in the list the next day.

The app uses the browser's local storage to store tasks as JSON.
